Projects

Here are some projects I've made both for class and just for fun. All of these projects are available on my Github.

WUSTLCourses: Course Browsing Platform

As an incoming tranfer student this past summer, I realized that there was no available alternative to my university's clunky course browsing platform. I set out to change this using Python to scrape the course data off of the internet and Django to create an intuitive web platform for browsing courses.

Check out the website!

WordCrunch: Blogging Social Network

This project was developed as the final project for CSE 330: Rapid Prototype Development and Creative Programming, which I took in Fall semester of 2017. The original concept was the create a social network site for bloggers similar to Tumblr but with an upvote/downvote system similar to Reddit's. This website was created using Django and had SQLite as a backend database.

Let's Chat: Instant Messenger Application

This project is an instant messaging application implemented using Node.JS, AJAX and Socket.IO. It allows users to send private messages, communicate in the lobby, and create their own chatrooms. In addition to sending plaintext messages, users can also style their messages and send images from the internet.
